Dart and Flutter static analysis

Build a resolved index. Query it from the CLI.

codegraph indexes symbols, callers, imports, provider reads, typed routes, and test relationships. It calculates impact, selects tests, and applies verified renames. If an edit cannot be proven complete, it refuses.

Unsafe edits return exit 3.
Rename refuses when symbol identity or the complete edit set cannot be proven. Affected-test plans expand when static analysis is uncertain.
  • Ambiguous identities return candidates.
  • External override chains block unsafe renames.
  • Dynamic or stale boundaries run more tests.
  • Every answer states its indexed scope.
